二、内边距 padding

作用：改变盒子里面内容位置
padding-left:px;左边内边距
padding-right:px;右边内边距
padding-left:10px;/*左内边距*/
padding-right:10px;/*右内边距*/
padding-top:10px;/*上内边距*/
padding-bottom:10px;
混合属性 padding:上 右 下 左; 四个属性 padding:0 0; 上下 左右

三、外边距 margin

尽量少用
margin-top:20px; 上外边距
margin-bottom:20px;下外边距
margin-right:auto; /*自动往左边走*/
margin-left:auto; /*自动往右边边走*/
特性:1、上下外边距会叠压； 外边距合并 以最大值来算;
2、父子级包含的时候子级的margin-top会传递给父级；（内边距替代外边距）
外边距复合：margin: top right bottom left;
混合属性：margin:上(top) 右（right） 下(bottom) 左(left);
margin:0 0 ;上下 左右
