1、html概念

超文本标记语言
< html> </html>

2、注意事项和编写规范

a.标记组成
< html>
b、成对标记
单标记< /> 双标记< > </>
< html></html>
c、标记有层级关系
< html>
< head></head>
< body></body>
</html>
d、标记有内容文本 或者是属性
< title>这是我的第一个网页</title>
< meta name="" content="" ></meta>

3、HTML的基本机构

doctype 网页文档的第一句 <!doctype html> 写死 W3Ch5 万维网联盟的规范
html 网页的根标签
head head标签里面大部分设置网页信息
body body标签里面大部分是可视化标签

4、特殊符号

  空格符
> 大于号
< 小于号

知识点

1、可视化标签 （标记） 只有可视化标记CSS样式才能改变它
2、非可视化标记 < span></span> < br/> < hr/> < img>
3、 注释标签

div 标记 层 通俗（容器） 一般用来布局用的
a 标记 超链接 href="跳转地址" target="_self" _self 本页面跳转 _blank 新窗口
img 图片标记
src* 属性用来设置图片的 url 数据 相对 绝对
alt 提供给手搜索引擎抓取的
table tr 行 行的colspan 列 rowspan
td 行里的格子
