1、CSS的概述

a、CSS（层叠样式表） 页面化妆师 对页面修饰
b、CSS layout (布局)
作用:它的作用让页面中的可视化标记变得漂亮

2、样式的类型

a、外联样式
引入方式:在head里面通过link 引入外部CSS文件 < link rel="stylesheet" href="css文件的路径" />
特性：拓展性强 多个页面使用 上线的之后 大多数
b、内部样式
引入方式：在head里面style < style> 修饰语法 < /style>
特性：单一页面使用 不可多个页面使用
c、内联样式
引入方式: 在标记里面写得 < p style=""></p> div span.. 都可以
特性：调试的页面时候使用它
优先级 a< b < c 就近原则

1、标签选择器 标签名+{} 一系列div都有 {}里面的属性
2、 ID选择器 #自定义id名{} 单一的名字才有 {}里面属性
3、class选择器 .自定义class名字 一系列带有class名字都有{}里面属性
id class命名的格式标准：
26个字母的组合 （见名知义） 不能纯数字
4、群组选择器 h1,h2,h3,h4,span,div,ul,li,dl,dd,dt{margin:0;padding:0;}
5、后代选择器 span p
6、伪类选择器
