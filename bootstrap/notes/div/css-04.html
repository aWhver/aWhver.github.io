二、高级标签 列表

1、无序列表 结构 < ul> 无序
< li>列表</li>
< li>列表数据</li>
</ul>
特性:a、默认情况小黑点
b、页面搜索引擎来看列表数据部分序列号
c、页面出现最多

三、样式的名字层级写得越详细 准确性越高

a.cc 不空格就是自己
a .cc 空格就是a下方小弟
d、ul li块级元素 有外边距
e、什么时候用 页面大部分风格相同部分
2、有序列表
结构 < ol> 有序
< li>有序的列表</li>
</ol>
特性：a、默认情况有序号 123...
b、页面搜索引擎来看数据分顺序
c、前面带有数字分类 有外边距
3、自定义列表
结构 < dl>内容块
< dt>内容标题</dt>
< dd>内容</dd>
</dl>
特性、 a、默认情况有内容标题 有内容
b、产品内容展示时候用到
C、有外边距
