用户注册登录 用到一些常用表单标签

一、form 在ie6以下 它就默认有外边距

属性 action="提交前端数据到后端接口" action="www.tanzhouedu.com"
表单提交的数据方式： method=""
method="" a:get(明文) 安全性不高 快
method="" b、post（密文） 安全性高 相对get有点慢

二、input

a、收集用户信息 type="什么类型收集数据"
b、提交按钮：< input type="submit" />
c、文本框：< input type="text" /> 样式默认背景是白色
d、密码框：< input type="password" />
e、单选按钮:性别< input type="radio" name="sex"/>男< input type="radio" name="sex"/>女 如果许多内容进行
选择 在radio标签里面 加相同的name="名字"
f、复选框：你是什么类型的人   < input type="checkbox" name="" disabled/>宅男
< input type="checkbox" name="" disabled/>宅女
< input type="checkbox" name="" checked/>外向

disabled 禁选 checked 默认选择
