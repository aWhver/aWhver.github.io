float-浮动问题---会造成元素父级高度塌陷
clear:left | right | both ；规定元素的某侧不允许浮动元素

解决办法

一、父级办法

  1. 加高 问题：扩展性不好
  2. inline-block 清浮动方法： 问题：margin:auto失效；
  3. overflow:hidden 清浮动方法； 问题：需要配合宽度
    overflow: 内容溢出
    1.scroll | auto | hidden（解决ie6下最小高度19px问题）；
    2.overflow-y:hidden;
    3.overflow-x:hidden
  4. after伪类 元素内部末尾添加内容 (时下主流)
    after{content:'';display:block;clear:both;}

二、给父级最后添加空标签

.clear{clear:both}
< div class="clear"></div>

三、其他

a、利用< br clear="all"/>
b、 课后了解hasLayout
c、利用伪类清除浮动 bat公司常用来清除的方法
1、before
2、after
兼容 ie8以上的版本浏览器
d、IE6下使用浮动 同时使用外边距 元素被放大2倍
解决办法：display:inline;
