新增标签：

结构标签：

< header></header> //定义页面头部的区域
< article></article> //定义页面独立的文章内容
< nav></nav> //定义导航
< section></section> //定义某一块区域
< aside></aside> //定义侧边栏
< footer></footer> //定义文档底部区域
< figure></figure> //定义一块独立的内容（通常放图片及描述）
< figcaption></figcaption>//定义figure的标题

特殊样式标签：

mark 突出显示文字
meter 刻度百分占比 有value属性 min max value表示占min至max的比率
low high 超出或低于变色
progress 进度条
ruby 内容+注释 rt 注释内容 rp 不支持时显示
wbr 长单词换行位置

兼容性不是很好地标签：

dialog //定义一个对话框 ff/chrome支持
bdi 文字摆放方式
details/summary 描述某部分的详情
< details>< summary>< /summary></details>
datalist
