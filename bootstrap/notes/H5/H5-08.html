video:

autoplay 自动播放（打开就播放）

controls 向用户显示播放控件

width 设置宽度

height 设置高度

loop 是否循环播放

preload 是否等加载完在播放

src url地址

poster 加载等待的画面图片

autobuffer 设置为浏览器缓冲方式 autoplay不设置才有效

video API

play(),pause(),load()

playbackRate 播放速度

全屏：element.webkitRequestFullscreen();webkit为兼容前缀

退出全屏：document.webkitCancelFullscreen();webkit为兼容前缀

w3c不需要前缀

audio:

currentTime 从开始到现在所用时间

duration 总时间

volume 0-1的音量相对值

muted 是否静音 true/false

pasued 是否暂停（只读）

ended 是否播放完毕（只读）

创建一个与音乐频次等长数组的方法

var voicehigh = new Unit8Array(analyser.frequencyBinCount)

将分析出来的音频数据添加到数组里面

analyser.getByteFrequencyData(voicehigh)
