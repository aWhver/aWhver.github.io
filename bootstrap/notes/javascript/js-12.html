面向对象：

一种开发模式, 所有的东西从对象出发

在函数执行之前加new关键字：

1.函数的内部会自动产生一个新对象

2.函数内部的this指向这个对象

3.函数默认返回这个对象

=的作用：

赋值：基础数据类型

引用：对象、函数

原型：prototype

只有构造函数有原型 构造函数.prototype

在写面向对象的时候注意：

私有的属性/方法，放在构造函数里

公有的属性/方法，放在原型里

instanceof :检测对象与构造函数在原型链上是否有关系

function fn(){};

var a = new fn();

a instanceof fn //true / false

hasOwnProperty() 检测对象自身下的属性

包装对象：

基础数据类型都有包装对象，不能存值

当某个基础类型在进行 . 操作的时候，会由他的包装对象来代替操作

完成之后，包装对象消失

每一次找的包装对象不一样

原型链：就近原则

描述对象和原型之间的关系

继承：

1.儿子继承爸爸

2.儿子的改动不影响爸爸
