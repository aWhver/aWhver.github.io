一、函数的定义：
1.匿名函数
function（）{} 不能单独写，会报错
2.有名函数
function fn(){}
调用：fn（）；自执行
二、函数表达式:
符号：~ - + （）
例如：（function（）{}） ~function（）{}
var a = function（）{} （）；第二个括号表示执行
意义：
1.节省变量
2.让函数内部的代码模块化，即内部代码独立
可以用外面的东西，外面的不能用里面的
三、函数定义与表达式区别：

函数表达式可以加括号自执行
