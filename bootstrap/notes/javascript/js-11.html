1.var xhr = new XMLHttpRequest() //创建一个ajax对象
2.xhr.open("get/post","文件",true)
请求的准备工作:
参数1通过什么方式请求：
1.get 发送的数据放在url里面
2.post 发送的数据放在头文件里面
参数2请求那个页面
参数3是否异步
3.xhr.send()
正式请求发送
4.onreadystatechange:事件，当请求的状态码发生改变的时候触发
readychange：请求的状态码
0：请求还没有建立（open执行之前）
1：请求建立了但还没有发送（open执行）
2：请求正式发送（send执行）
3：请求已经受理，有一部分数据可以用了，但还没有完全处理完
4：请求完成，所有数据都处理完
responseText:返回的数据
eval（）把字符串当成js代码来读变成数组
