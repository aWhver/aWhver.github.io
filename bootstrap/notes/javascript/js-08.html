字符串：

（以下对象指字符串） 长度属性（length）只能读，不能改
字符串里的某一个字符 只能读，不能改
charAt（num） 内置的一个方法 找到字符串下标为num的字符
charCodeAt（num） 找到某个位置字符对应的ascll
String.fromCharCode(ascll) 把ascll转换成字符
对象.indexOf（）//-1 找不到
对象.indexOf（‘字符’）在对象里找到字符所在的位置空格也占位置 空字符为0
对象.indexOf（‘字符’，num）从第几位开始找 默认从0
对象.lastIndexOf(“字符”) 从后往前找 属性与indexOf一样

截取字符串：

对象.substr（num）//从第num位开始截取所有字符串
对象.substr（num1，num2）//从第num1位开始截取到下标为num2-1的位置
substr是主流
对象.slice（num）//从第num位开始截取所有字符串
对象.slice（num1，num2）num位置不能交换
对象.slice（-1）//最后一位-1 最后2位-2....

字母转换：

toUpperCase() //转换成大写
toLowerCase() //转换成小写

数组：

长度属性（length）能读，能改，内容亦可
var arr=【】；以字面量的方式创建数组
var arr=new Array（）；以对象的形式创建数组 只一个数字就是长度
数组.push（） 往数组后面添加内容
数组.unshift() 往数组前面添加内容
数组.pop() 删除数组的最后一位
数组.shift(）删除数组的第0位
数组.splice(num1,num2) 从第num1（下标）位开始 删除num2位
数组.splice(num1,num2，内容) 从第num1位开始 删除num2位 替换内容
数组.sort();数组排序 默认从小到大
数组.sort(function（x，y）{
return x-y //正值 小到大
return y-x //负值 大到小
})
数字1.concat（数组2，数组3）// 数组拼接
字符串.split（“”）//把字符串以空字符切割成数组
join("") 把切割成数组的字符串连接成字符串
