一、算术运算符：

+ - * / %(取余)
字符串拼接： +： 2个数字做加法运算 如果有字符串，字符串拼接
document.write('< div>'+a+'</div>')；
-： 2个数字做减法运算 如果有字符串不是数字，NaN 不是数字 是number类型
如果是数字，把字符串的数字转成number，在做运算
*： 2个数字做乘法运算，如果是数字，把字符串的数字转成number在做运算
== {运算符}优先级高于 = {赋值}
